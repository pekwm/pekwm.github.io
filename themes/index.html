Contributing Themes

To include your theme create a pull request against the pekwm-theme-index repository with a JSON file describing your theme.

Managing Themes

pekwm themes on this page can be managed using the pekwm_theme command shipped with pekwm-0.2.0 and later versions.

install themes by running: 

$ pekwm_theme install name

list available themes by running: 

$ pekwm_theme install name

show a theme before installing by running: 

$ pekwm_theme show name
