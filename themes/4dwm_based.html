4dwm_based theme

Author
Paulo Silva
Homepage
https://notabug.org/nitrofurano/pekwmthemes
This is just a PekWM theme superficially based on SGI Irix 4DWM window manager - the button, decoration bevel and window title text arrangement is different from the original, and the button icons changes on mouse over and pressed behaviours (the original 4DWM ones doesn't do this).

Install

$ pekwm_theme install 4dwm_based
